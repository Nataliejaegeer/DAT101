Discussion / Answers

  1. Discuss the behavior of part 1, What happens when the wake-up time is not 7 and what happens when it is exactly 7?

    2. If wake-up time is not 7, nothing shows. If it is 7, the bus message prints.

  2. Same as question #1. Discuss the behavior of part 2, What happens when the wake-up time is not 7 and what happens when it is exactly 7?

    3. If wake-up time is 7, I can take the bus. Otherwise I take the car.

  3. Discuss the behavior of part 3, what happens when wake time is less than 7 or greater than 8?

    4. If wake-up time is 8, I take the train. If it is 7, I take the bus. If it is less than 7 or greater than 8, I take the car.

  4. Discuss the behavior for part 4, what happens if the number is exactly 0, is zero positive or negative?

    5. Zero is neither positive nor negative. The code prints "Zero".

  5. Part 5: Extended number check

    6. The code now prints "Positive", "Negative" or "Zero" depending on the value of the number.

  6. Part 6: Photo upload

    7. If image is smaller than 4MP, it says "The image is too small". Otherwise it prints "Thank you".

  7. Part 7: Photo size limits

    8. Images smaller than 4MP are too small, images 6MP or larger are too big, otherwise it prints "Thank you".

  8. Part 8: Vitamin D check

    9. If the month name contains "r", it says "You must take vitamin D", else "You do not need to take vitamin D".

  9. Part 9: Days in month

    10. The code prints how many days each month has. February has 28, April, June, September, November have 30, others 31.

  10. Part 10: Gallery status

    11. The gallery is closed in March and May, open next door in April, and otherwise open normally.
