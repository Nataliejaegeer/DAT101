1. What is an array?

    2. An array is a collection of values stored in a single variable. Each value has an index, which starts at 0. Arrays are useful when we want to store multiple values in one place.

  2. What does it mean to "hardcode" when we talk about programming?

    3. Hardcoding means writing values directly into the code instead of getting them from input or calculating them. For example, creating an array with fixed numbers instead of generating them.

  3. Explain in plain text what happens in task 3, and how we can traverse the array when we don't know the length.

    4. In task 3, we take a sentence and split it into an array of words. Then we loop through the array and print each word with its index. When we don't know the length, we can use a loop with array.length so it automatically stops at the end.

  4. Why is it beneficial to store instances of a class in arrays?

    5. It is useful because we can store multiple objects in one place and easily loop through them. For example, storing books in an array makes it easier to print or manage them without writing separate variables for each one.

  5. Explain the EWeekDays object and how it works.

    6. The EWeekDays object contains different types of days like weekdays and weekends. The value attribute can be used to store extra information about each day. WeekDay represents a single day, while Workdays and Weekends represent groups of days. If we wanted to create WorkoutDays, it could look like an array with selected days like Monday, Wednesday, and Thursday.

  6. Explain how a callback function works in sorting.

    7. A callback function is a function that is passed into another function. In sorting, the callback decides how two values should be compared. It returns a number that tells if one value should come before or after another.

  7. What happens in the sorting routines?

    8. The sorting function compares elements two at a time. Based on the result from the callback, it decides the order. If the result is negative, the first element comes first. If it is positive, the second element comes first.

  8. Explain what the first element of a two-dimensional array contains.

    9. The first element of a two-dimensional array is another array. It represents the first row, and inside it are the columns.
